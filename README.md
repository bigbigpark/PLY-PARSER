# Ply Parser

convert *.ply to *.ply

<br/>

properties : <br/>
float32 x <br/>
float32 y <br/>
float32 z <br/>
float32 timestamp <br/>


## How to use

clone this repo. <br/>

~~~bash
$ git clone https://github.com/bigbigpark/PLY-PARSER.git
~~~

build and run

~~~bash
$ cd PLY-PARSER
$ mkdir build
$ cmake ../
$ make
~~~

Specify the folder <br/>

~~~bash
$ ./main sangam
~~~
